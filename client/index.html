Generative-AI, without the fuss.

Use tailgate to build custom, generative-AI features into your website without setting up complex backend infrastructure. Generate text, images, and let users ask questions over custom data.

Examples:

  • Meal Idea
  • Menus QA
  • Recipe Generator

For setup instructions and documentation, see our Github. For any questions or assistance, email us here. We'd love to hear why you are interested in using tailgate, and how we can help!

Basic Usage

Simply import the javascript client from a CDN:

<script src="https://cdn.jsdelivr.net/gh/sethkimmel3/tailgate/client/cdn/tailgate.js"></script>

Decorate some elements:

<p data-gentag data-prompt="A haiku about a tailgate."></p>
<img data-gentag data-prompt="A tailgate." /><

And instantiate the library with your API key:

<script> new tailgate('example-public-key');></script>
